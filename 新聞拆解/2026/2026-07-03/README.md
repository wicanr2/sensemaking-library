## 今日共同觀察

今天三篇都在講同一件事：AI 競爭正在從「誰的模型 / 晶片比較強」下沉到「誰控制供應鏈選項」。聯發科被重估，是因為客製化晶片需求可能改變營收結構；日月光被重估，是因為先進封裝與測試變成 AI 交付瓶頸；Anthropic 洽談 Samsung，則是模型公司試著把算力供給選項往自己手裡收。

更抽象地說，AI 產業進入第二階段後，最值錢的不只是單點性能，而是「能否把稀缺資源排成可交付系統」：設計、製程、封裝、測試、伺服器、電力、客戶長約都在同一條因果鏈裡。
